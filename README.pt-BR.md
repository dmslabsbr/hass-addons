## Sobre

Este repositório reúne add-ons desenvolvidos para uso no [Home Assistant](https://www.home-assistant.io/), permitindo integrar dispositivos e serviços de forma centralizada no seu painel.

**Mantenedor:** Daniel — suport@neoage.com.br

---

## Requisitos

- [Home Assistant](https://www.home-assistant.io/) (instalação com **Supervisor**)
- Acesso à interface do Supervisor para adicionar repositórios de terceiros

---

## Instalação

1. No Home Assistant, acesse **Configurações** → **Add-ons** → **Add-on Store**.
2. Clique no menu (três pontos no canto superior direito) e escolha **Repositórios**.
3. Adicione o repositório com a URL:

   ```text
   https://github.com/dmslabsbr/hass-addons
   ```

4. Clique em **Adicionar** e aguarde o carregamento.
5. Os add-ons deste repositório aparecerão na loja; instale e configure conforme cada add-on.

Instruções oficiais: [Installing third-party add-ons](https://www.home-assistant.io/common-tasks/os#installing-third-party-add-ons).
