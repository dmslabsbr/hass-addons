## About

This repository hosts add-ons developed for [Home Assistant](https://www.home-assistant.io/), enabling you to integrate devices and services in a single dashboard.

**Maintainer:** Daniel — suport@neoage.com.br

---

## Requirements

- [Home Assistant](https://www.home-assistant.io/) (installation with **Supervisor**)
- Access to the Supervisor UI to add third-party repositories

---

## Installation

1. In Home Assistant, go to **Settings** → **Add-ons** → **Add-on Store**.
2. Open the menu (three dots in the top-right corner) and select **Repositories**.
3. Add this repository using the URL:

   ```text
   https://github.com/dmslabsbr/hass-addons
   ```

4. Click **Add** and wait for it to load.
5. Add-ons from this repository will appear in the store; install and configure each one as needed.

Official instructions: [Installing third-party add-ons](https://www.home-assistant.io/common-tasks/os#installing-third-party-add-ons).
